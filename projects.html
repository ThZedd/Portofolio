Zoopolis

Kotlin - Java - Python

Zoopolis is an interactive mobile app that serves as the ultimate digital guide for zoo visitors. Designed to solve on-site navigation and information gaps, it features real-time location tracking, educational resources, and a gamified point-accumulation system to maximize visitor engagement and enjoyment.

DarkLeague

JavaScript - HTML - CSS

DarkLeague is an all-in-one web platform for trading card collectors and competitors. It features a digital catalog for collection management, custom user preferences, and a detailed search engine for card prices and stats. Its standout feature is a robust tournament management tool that streamlines scheduling, player registration, and match tracking, ensuring a complete and interactive experience for the community.
